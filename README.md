Usage & Customisation

You’re welcome to use this template for your personal portfolio. To customise:

Update the content in each section (About Me, Skills, Projects) to reflect your own journey.

Replace the project links with your own GitHub repositories or live demos.

Change colour schemes, fonts, layout details to match your style.

Ensure you include your own contact details (email, GitHub, LinkedIn).

If you add new sections (Blog, Testimonials, etc), update the navigation so everything stays simple and intuitive.
